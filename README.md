# Ultimate FX ML Trading Engine 🚀

## Vision
Community-driven ML engine for Forex trading with institutional-grade robustness.

## Objectives
- High Sharpe Ratio: maximize returns with minimal volatility
- Superior Win Rate: accuracy over noise
- Regime Adaptability: thrive in both trending and ranging markets
- Professional Visualizations: equity curves, drawdown maps, signal heatmaps

## Tech Stack
- Models: LSTM, XGBoost, Reinforcement Learning (PPO/DQN)
- Data: 1m/5m/1h intervals for majors (EUR/USD, USD/JPY, GBP/JPY, XAU/USD, XAU/AUD)
- Backtesting: VectorBT, Backtrader
- Dashboard: Plotly, Streamlit
